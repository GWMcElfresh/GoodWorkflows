GoodWorkflows Synthetic Tabulation Walkthrough

2026-04-14

Purpose

This notebook is the repository-local companion to the MkDocs vignette. It uses only seeded synthetic data, so it is safe to render on any machine and does not require access to Prime-seq, LabKey, or sensitive trial tables.

The fixture bundle is generated by:

Rscript tests/fixtures/simulate_trial_data.R \
  --output-dir tests/fixtures/synthetic_trial_data \
  --target all \
  --seed 20260414

Workflow mapping

  • ingest_tabulate consumes metadata-like rows and emits subjectIdTable.csv.
  • ingest_export emits a 10x-like count directory with matrix.mtx, features.tsv, barcodes.tsv, and obs_meta.csv.
  • The same synthetic fixture bundle is used in smoke tests and documentation examples.
